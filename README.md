# Nassau_Candy_Profitability_Analysis

## Product Line Profitability and Margin Performance  Analysis for Nassau candy distributor using Excel, SQL and Power BI

## Tools used 
- Microsoft Excel - EDA ,KPI calculations,Pivot tables
- SQL (SQLite) - Data validation and aggregation queries
- Microsoft Power BI - Interactive Dashboard
  
## Key findings
- Chocolate Division contributes 92.88% of total revenue
- Top 5 Wonka bars products generate 95% of total profit
- Kazookles identified as highest margin risk at 7.69%
- 16 states drive 80% of total profit geographically
  
## Files in this repositary
- Nassaucandy_clean.xlsx - cleaned dataset with KPI calculations with segmented Analysis
- Nassaucandy_Queries.sql - All SQL used in the analysis
- Nassaucandy_Dashboard.pbix - Power BI Dashboard file
- Research_Paper.pdf - Full research paper and Analytics report
